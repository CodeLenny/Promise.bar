Promise.bar » Index » Progress

Class: Progress

Defined in: PromiseBar.coffee

Overview

An individual item to be used with PromiseBar.

Property Summary

(Number) done
the count of completed items
(Progress) parent
set to another progress bar if this element appears under another element.
(String) indent Readonly
the proper number of spaces to indent progress bars that are in a hierarchy.
(String) rawLabel Readonly
a label for this progress bar.
(String) label Readonly
a label for this progress bar, optionally padded to make all labels the same length.
(Number) total Readonly
the total number of items.
(String) percent Readonly
the percent of completed items.
(Array<Progress>) children Readonly
the children progress bars that are descendants of this one.

  • # (void) tick() Called when one of the Promise items is resolved.
